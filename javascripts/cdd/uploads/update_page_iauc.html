    • Instructions

    How to use:

    Youtube

    Username and Password

    The username and password must be obtained in order to send data to the Jan Japan Purchase System.
    However, password is not required, if you only choose Input GSS as a sending option.Note

    Purchase Country

    The purchase country must be set according to the username. Otherwise, you will not be able to send data to jjpurchase system.

    Auction Date

    Auction date is automatically set to tomorrow when you first start iauc. But you can manually enter any desired date to match the selected auction.
    Any change made to the date manually is saved until the next day or change.

    Change List:

    Change List button allows you to set the default list you want a lot you place a bid for gets added in.
    Selecting A will cause a lot to be added to list A, if it is already not listed to A. In case the lot is already in A list, no action will be taken

    LOT PRICE

    LOT and PRICE are is for display only. Simply displays the last sent lot and bid price.

    Status

    Status will show the current status of data as request or success.
    Status will remain as request, if an error occurs. Please, contact the developer in such cases.

    Reset Counter

    There are two counters: JDB and GSS. JDB stands for Jan Database and GSS is Google Spreadsheet.
    The numbers will increase depending on your selections in options panel. Reset Counter buttons set the numbers to their left to 0.
    The counters are for display purposes only and do not effect the process of sending or retrieving data from the respective datasource

    Sort Buttons and more

    Sort By Lot sorts the lots in the current view by lot numbers.
    Sort A-Z is used to sort lots alphabetically
    Auction Filter is is used to select auctions.
    *You need to use a proper desktop version of Chrome browser for the functions to work.
    Show Prices is used to display all existing bids for all lots in the current view. The data is fetched from the jjpurchase system. Greenish color boxes are displayed below the lot thumbnails. The box is called: Price Manager and allows you to change sent bid price directly. You can only change the price for your own bids or markets under your responsibility. If you try to change a different market price, you will receive a warning message and the action will be dismissed.

    A B C D

    A, B, C and D buttons are used to add all the lots in the current view to that list

    Add Price Fields

    It is an emergency button that can be used to make the price and remarks fields to be added, in case they fail to get added automatically.

    'More' Button

    Market Price is used to generate a sold price report cased on the list of cars from your list. The report is generated and downloaded as a CSV (comma separated values) file, which can be easily opened with Excel or similar software. The sold price and bid price are automatically compared and colored green or red accordingly. You must be in My List view to get reliable results. An attempt to generate a report from Search Cars view may result in logout from iauc.
    Make sure to set the Auction Date equal to that of the list you are trying to generate report for.
    *Caution: The report depends on your username, password and the auction date to be set correctly. If you made a mistake, you must refresh the page to reset the values. However, this limitation will be removed soon and notification shown accordingly.
    You can generate one report per iauc page. If you have more pages, go to the "Next" page and repeat.

    Auto Pick My Cars searches for cars with matching year and model and inputs bid price automatically. The price will be shown in deep pink color

    Send all with 0 price

    The option is made for Japan users only. Pressing the button will send all the lots in the current view by 0 price to JDB or/and GSS depending on your choice. Selecting Input GSS will request you to load data before sending. The button used for loading data will flash for easy navigation.

    Options/Quick Settings

    When you click Options button Quick Settings panel is displayed
    Input JDB checkbox is for sending data to jjpurchase system.

    Input GSS checkbox is used to send data to Google Spreadsheet. You need to select your preferred spreadsheet from drive when the dialog opens. The next window will let you choose the worksheet name you want to send your data to.

    Popup v2 checkbox is used to enable popup window with full features, including its own copy of DLM.


    Auto Sayuri is helpful especially while working on details page view
    Show Type helps easily see the chassis type in thumbnail view, which hides the type by default.

    Autostart ON/OFF
    You can turn autostart off, when you do not need the DLM to start while viewing IAUC. Right click the extension icon and choose options. Uncheck the checkbox and press SAVE button.
    The checkbox must be checked and status saved for the extension to be used.

    Working through Detail Page
    You can input bids on this view as well.
    There are a few differences though. It is better to use on your keyboard rather than Back and Next buttons.
    Arrow keys will cause the bid input fields appear automatically.
